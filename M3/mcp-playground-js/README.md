# MCP Playground (JS)

## docs

- https://github.com/modelcontextprotocol/typescript-sdk

## install

`npm install`

## test (MCP inspector)

`npm test`
`npm run test:cli`
see `package.json`

## config

```json
{
  "mcpServers": {
    "mcp-playground": {
      "command": "node server.js"
    }
  }
}
```
